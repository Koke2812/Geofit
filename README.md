Geofit es una aplicación móvil desarrollada para ayudar a los usuarios a mejorar sus hábitos saludables mediante el control de actividad física, nutrición y seguimiento personal diario.

La aplicación ha sido diseñada con una interfaz sencilla, intuitiva y fácil de utilizar, permitiendo que cualquier usuario pueda acceder rápidamente a sus funciones sin necesidad de conocimientos técnicos.

---

# Objetivo de la aplicación

El objetivo principal de Geofit es proporcionar una herramienta práctica y accesible para gestionar información relacionada con la salud y el bienestar personal desde un dispositivo móvil Android.

La aplicación permite centralizar diferentes aspectos relacionados con el control físico y nutricional en un único entorno cómodo y organizado.

---

# Funcionalidades principales

Actualmente Geofit incluye funcionalidades como:

- Registro y seguimiento de actividad física
- Gestión nutricional
- Almacenamiento local de información
- Funcionamiento offline
- Interfaz adaptada para dispositivos Android
- Organización sencilla de datos personales

---

# Tecnologías utilizadas

El proyecto ha sido desarrollado utilizando las siguientes tecnologías:

- Flutter
- Dart
- SQLite
- Android SDK

---

# Compatibilidad

Geofit está orientada principalmente a dispositivos Android y puede ejecutarse en la mayoría de teléfonos móviles actuales compatibles con aplicaciones APK.
